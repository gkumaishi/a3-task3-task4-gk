Section 2: Purple urchin counts by site and year

1) Create a stored subset of the urchins data

-Filter to only include observations for purple urchins -Convert the date to class “Date” -Create a new column that only contains the year -Create a new column called site_full that contains the full name for each site

purple_urchins <- urchins %>% 
  filter(common_name == "Purple Urchin") %>% 
  mutate(date_new = mdy(date)) %>%
  mutate(year = year(date_new)) %>% 
  mutate(full_site_name = case_when( 
    site == "NAPL" ~ "Naples Reef", 
    site == "AQUE" ~ "Arroyo Quemado Reef",
    site == "CARP" ~ "Carpinteria Reef",
    site == "MOHK" ~ "Mohawk Reef",
    site == "IVEE" ~ "Isla Vista Reef"))
